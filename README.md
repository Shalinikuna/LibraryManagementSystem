# Library Management System (Spring Boot)

Backend REST API to manage books and users in a library.

## Features
- Add book
- Get book
- Update book
- Delete book
- Structured layered architecture

## Tech Stack
Java, Spring Boot, REST APIs, JPA, MySQL, Maven

## Run Locally
1. Clone repo
2. Open in IDE
3. Configure DB in application.properties
4. Run main application
5. Test APIs using Postman
